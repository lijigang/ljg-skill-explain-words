# ljg-skill-explain-words

单词灵魂解剖师 (Word Soul Master) — 一个 [Claude Code](https://docs.anthropic.com/en/docs/claude-code) Skill，深度解构英文单词的词源、核心语义、语感光谱与语义拓扑，生成博物馆级 HTML 卡片。

## 功能

- 词源拆解：拉丁/希腊词根 + 同源词网络
- 核心语义：原始画面 + 核心意象公式
- 语感辨析：易混淆词对比
- 语义拓扑：Mermaid 可视化图谱
- 一语道破：中英双语哲学金句
- 输出精美 HTML 卡片，浏览器直接打开

## 安装

```bash
git clone https://github.com/lijigang/ljg-skill-explain-words.git ~/.claude/skills/ljg-explain-words
```

## 使用

在 Claude Code 中输入：

```
/ljg-explain-words Serendipity
```

## 输出示例

生成一张包含以下模块的 HTML 卡片：

- **CORE MEANING** — 核心语义与原始画面
- **ETYMOLOGY** — 词根拆解与同源词
- **NUANCE & CONTEXT** — 语感辨析与例句
- **SEMANTIC TOPOLOGY** — Mermaid 语义拓扑图
- **EPIPHANY** — 一语道破金句
