# ljg-skill-explain-words

单词灵魂解剖师 (Word Soul Master) — 一个 [Claude Code](https://docs.anthropic.com/en/docs/claude-code) Skill，深度解构英文单词的核心语义，直击词的灵魂。

## 功能

- 核心语义：原始画面 + 核心意象公式 + 深层解释
- 一语道破：中英双语哲学金句
- 直接在对话中 Markdown 输出，轻量无依赖

## 安装

```bash
/plugin marketplace add lijigang/ljg-skill-explain-words
/plugin install ljg-explain-words
```

## 使用

在 Claude Code 中输入：

```
/ljg-explain-words Serendipity
```

## 输出示例

```
## Catharsis  /kəˈθɑːrsɪs/  宣泄；净化

原始画面：一个人蹲在河边，用清水冲洗伤口上的脓血...

核心意象：积压的污浊 + 强烈的排出 + 排出后的洁净 = 净化

（深层解释...）

> "Catharsis is not the avoidance of pain, but the completion of it.
> 净化不是绕开痛苦，而是把痛苦走完。"
```
